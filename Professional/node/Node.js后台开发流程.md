# Node.js后台开发流程

###### Node.js + espress + MongoDB(mongoose)

#### 1、创建框架

```
express -e miaomiaoserver
```

参数：-e   使用ejs

#### 2、安装依赖

```
cd miaomiaoserver

npm i
```

#### 3、运行服务

```
npm start
```

##### 4、创建数据库文件夹db，启动Mongo数据库

```
mongod
```

#### 5、邮箱验证

##### 模块：nodeMailer：
